لوحة تحكم المدير (Admin Dashboard)
الصفحة الرئيسية (Dashboard)
الصفحة الرئيسية للمدير تعرض إحصائيات ومعلومات مهمة مثل:

عدد الطلاب المسجلين

عدد المعلمين العاملين

عدد الامتحانات المجدولة

عدد الدفعات المُسددة

التنبيهات العامة

أحدث الأنشطة في النظام

المعلمين (Teachers)
إدارة المعلمين داخل السنتر، مع إمكانية:

إضافة معلمين جدد

عرض وتعديل بيانات المعلمين

تفعيل أو تعطيل حساباتهم

أولياء الأمور (Parents)
إدارة حسابات أولياء الأمور، مع إمكانية:

ربط أولياء الأمور بأبنائهم

عرض معلومات التواصل

إرسال إشعارات مباشرة لهم

الطلاب (Students)
إدارة بيانات الطلاب، تشمل:

إضافة أو تعديل بيانات الطلاب

تعيين الصفوف والمجموعات الدراسية

عرض تاريخ الحضور، الدرجات، والرسوم

المراحل الدراسية (Grades)
تعريف المراحل التعليمية الأساسية مثل:

المرحلة الابتدائية

المرحلة الإعدادية

المرحلة الثانوية

الصفوف الدراسية (Classes)
تعريف الصفوف داخل كل مرحلة، مثل:

الصف الأول الابتدائي، الصف الثاني الابتدائي

الصف الأول الإعدادي، الصف الثاني الإعدادي

الصف الأول الثانوي، الصف الثاني الثانوي

المجموعات الدراسية (Sections)
تنظيم الطلاب إلى مجموعات داخل الصفوف لسهولة الإدارة، مع إمكانية:

تحديد المعلم المسؤول عن المجموعة

تعيين مواعيد الحصص الدراسية

الوحدات الدراسية (Units)
تقسيم المقررات إلى وحدات صغيرة، تُسهل متابعة تقدم الطلاب في المادة التعليمية.

الدروس (Lessons)
إدارة المواد الدراسية والمحتوى التعليمي داخل كل وحدة، بما في ذلك:

ملفات الدروس

الفيديوهات التعليمية

الأنشطة التفاعلية

الحضور والغياب (Attendance)
متابعة حضور الطلاب بشكل يومي، مع إمكانية:

تسجيل الغياب والأعذار

إصدار تقارير يومية أو شهرية

الامتحانات (Exams)
إدارة الامتحانات النهائية لكل مادة، وتسجيل درجات الطلاب.

الكويزات (Quizzes)
إنشاء اختبارات قصيرة لقياس الفهم المستمر، مع تصحيح أوتوماتيكي أو يدوي.

الرسوم الدراسية (Fees)
تسجيل الدفعات المالية للطلاب، وتشمل:

تحديد قيمة الرسوم

تسجيل الدفع

إصدار إيصالات إلكترونية

الإعلانات (Announcements)
نشر الإعلانات العامة والخاصة مثل:

مواعيد الامتحانات

تعليمات الدخول والخروج

أنشطة مدرسية

التقارير (Reports)
إصدار تقارير شاملة عن كل ما يتعلق بالطلاب، مثل:

تقارير الأداء الدراسي

تقارير الحضور

تقارير الدفع

المكتبة (Library)
مستودع رقمي يشمل:

ملفات دراسية

كتب إلكترونية

نماذج امتحانات سابقة

الإعدادات (Settings)
إعدادات النظام العامة مثل:

بيانات السنتر

إعدادات البريد الإلكتروني والإشعارات

إدارة الصلاحيات

لوحة تحكم المعلم (Teacher Dashboard)
الصفحة الرئيسية (Dashboard)
ملخص لأنشطة المعلم، مثل:

عدد الطلاب الذين يدرّسهم

عدد المجموعات الدراسية

جدول الحصص

آخر الواجبات والامتحانات

المجموعات الدراسية (Sections)
عرض المجموعات التي يدرّسها المعلم، مع إمكانية:

تسجيل الحضور

متابعة الدروس

تقييم الطلاب

الحضور والغياب (Attendance)
تسجيل حضور وغياب الطلاب في كل حصة بشكل يومي.

التقارير (Reports)
عرض تقارير تفصيلية تشمل:

تقارير الحضور

تقارير درجات الامتحانات

تقارير الكويزات

تقارير الدفعات للطلاب

الملف الشخصي (Profile)
عرض وتعديل بيانات المعلم الشخصية، مثل:

الاسم والبريد الإلكتروني

كلمة المرور

المواد التي يقوم بتدريسها

لوحة تحكم ولي الأمر (Parent Dashboard)
الصفحة الرئيسية (Dashboard)
تعرض لمحة سريعة عن حالة الأبناء المسجلين، مثل:

عدد الأبناء في السنتر

أحدث الإشعارات المتعلقة بهم

الأبناء (Children)
عرض معلومات كل ابن/ابنة، مع إمكانية:

متابعة صفوفهم ومواعيدهم

الاطلاع على تقدمهم الدراسي

البحث عن الحضور (Attendance Search)
إمكانية البحث عن سجل حضور أبنائهم في تواريخ محددة.

تقرير الحضور (Attendance Report)
تقرير شامل عن حضور وغياب الأبناء مع توضيح أيام الغياب وأسبابه.

تقرير الامتحانات (Exams Report)
عرض درجات الامتحانات لكل مادة بشكل مفصل.

تقرير الكويزات (Quizzes Report)
عرض نتائج الكويزات القصيرة التي خضع لها الأبناء.

تقرير الرسوم (Fees Report)
عرض الرسوم المستحقة والمدفوعة، وإمكانية طباعة الفواتير.

الملف الشخصي (Profile)
عرض وتحديث بيانات ولي الأمر، مثل:

البريد الإلكتروني

رقم الهاتف

كلمة المرور

لوحة تحكم الطالب (Student Dashboard)
الصفحة الرئيسية (Dashboard)
عرض معلومات مهمة مثل:

عدد المواد المسجل بها

أحدث الأنشطة والواجبات

تنبيهات مهمة

الامتحانات (Exams)
عرض نتائج الامتحانات الرسمية مع تقييمات المعلمين.

الكويزات (Quizzes)
عرض نتائج الكويزات السريعة مع التعليقات عليها.

الرسوم الدراسية (Fees)
عرض الرسوم الدراسية المطلوبة والمدفوعة، مع إمكانية عرض تفاصيل الدفعات.

الحضور والغياب (Attendance)
عرض سجل الحضور والغياب الخاص به، مع توضيح الأسباب.

المكتبة (Library)
الوصول إلى مكتبة المواد الدراسية:

كتب إلكترونية
